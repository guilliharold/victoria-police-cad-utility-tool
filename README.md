# 🚔 Victoria Police CAD Tool

A web-based tool for generating realistic Victoria Police CAD lineups for use in [MissionChief-Australia](https://www.missionchief-australia.com/) and similar emergency service simulation games.

No installation, no login, no fuss — just open it in your browser and go.

## 🚀 Getting Started

The tool runs as a simple three-step wizard:

```
1️⃣  Pick your station  →  2️⃣  Choose services  →  3️⃣  Get your lineup
```

Just work through each step from left to right. You can always go back and change your selections at any point.

---

## 1️⃣ Step 1 — Select a Station

Use the three dropdowns to locate your station:

1. **Region** — choose from North West Metro, Southern Metro, Eastern, or Western
2. **Division** — the divisional area within that region (e.g. Goulburn Valley, Bendigo)
3. **Station** — the specific police station

Once a station is selected, two more options appear:

### 🏷️ Station Classification

This controls how many units get generated by default. The tool pre-fills this from the station database, but you can override it if needed. See [Station Classifications](#-station-classifications) for a full breakdown.

### 👮 Supervisor Required

- **Yes** — includes Command & Supervision units (Station SGT, District Patrol SGT, and senior ranks where applicable)
- **No** — units only, no supervision entries

When finished, click **Next — Select Services →**

---

## 2️⃣ Step 2 — Select Services

Tick the services you want included at this station. You can select as many or as few as you like.

> 💡 **Tip:** Select only the services that realistically operate from this station. A small rural station probably only needs Station Cars and Divisional Vans, while a major metropolitan station might run HWP, CIU, SOCIT and more.

### 🏍️ Solo Motorcycle Options

If you select **Highway Patrol** or **State Highway Patrol**, an extra option card will appear immediately after it in the grid:

- **Highway Patrol Solo** — adds station-based solo motorcycle units (station code prefix, 600–609)
- **State Highway Patrol Solo** — adds TRF-prefix solo motorcycle units (TRF600–609)

These are off by default since solo motorcycles are less common than patrol cars.

When ready, click **Generate Lineup →**

---

## 3️⃣ Step 3 — Your Lineup

Your generated lineup appears here, grouped by service type. Each unit shows:

- 📻 **Callsign** — e.g. `ESP311`, `TRF614`, `POR601`
- 📋 **Description** — the unit's role

### 🎚️ Adjusting Unit Counts

Each scalable service (Station Cars, Divisional Vans, Highway Patrol, State Highway Patrol, CIU, PORT, RRU) has a **slider** you can drag to increase or decrease the number of units shown.

> 💡 **How the slider works:** The tool always tries to keep a balanced spread across morning (0700), afternoon (1500) and night (2300) shifts. For Highway Patrol, it also maintains roughly one unmarked Q Car for every two marked cars — no matter where you set the slider.

### 🔗 Station Support Links

If the station has PSA, HWP and CIU codes configured in the database, a **Station Support Links** card will appear showing which stations provide those services.

### 📋 Export / Copy

At the bottom of the output is a formatted plain-text version of the lineup. Click **Copy to Clipboard** to copy it, then paste it wherever you need it — MissionChief, a notes app, Discord, anywhere.

> 🔄 The export updates live as you adjust sliders, so it always reflects what's on screen.

## 🏛️ Station Classifications

| Classification | Label | Description |
|---|---|---|
| `metro_24` | Metropolitan (24 Hours) | High-density urban station, continuous 24/7 coverage, high call volume — e.g. Box Hill, Dandenong, Melbourne West |
| `metro_non24` | Metropolitan (Non-24 Hours) | Metro station without a continuous 24hr public counter; patrol coverage maintained via nearby hubs — e.g. Brunswick, Rowville, Springvale |
| `regional_24` | Regional (24 Hours) | Key regional hub, continuous 24/7 coverage across a large geographic area — e.g. Bendigo, Geelong, Shepparton |
| `regional_non24` | Regional (Non-24 Hours) | Regional or rural station; coverage via mobile patrols and nearby hub support — e.g. Bright, Echuca, Yarrawonga |
| `regional_single` | Regional (Single Member) | Single-officer station in a small rural town, one vehicle only — e.g. Dookie, Katamatite, Sea Lake |

**Classification affects two things:**
1. 🔢 The default number of units generated per service
2. 👮 Which supervision units are included

| Classification | Supervision units |
|---|---|
| Metropolitan (24 Hours) | 250, 251, 252, 260 (Senior SGT), 265 (Div Supervisor S/SGT) |
| Metropolitan (Non-24 Hours) | 250, 251 only |
| Regional (24 Hours) | 250, 251, 252, 260, 265 |
| Regional (Non-24 Hours) | 250, 251 only |
| Regional (Single Member) | 251 only — no on-site Station SGT |

## 📡 Callsign Number Guide

### Station Code Format

```
[Region letter] + [Station abbreviation]

N = North West Metro      S = Southern Metro
E = Eastern               W = Western

Examples:
  E + SP  →  ESP    Shepparton
  W + BI  →  WBI    Bendigo
  W + GL  →  WGL    Geelong
  N + MN  →  NMN    Melbourne North
```

### Shift Number Convention

For **Station Cars** and **Divisional Vans**, the trailing digit tells you the shift:

| Suffix | Shift | Example |
|---|---|---|
| `x07` | Morning (0700 start) | `ESP207`, `ESP307` |
| `x03` | Afternoon (1500 start) | `ESP203`, `ESP303` |
| `x11` | Night (2300 start) | `ESP211`, `ESP311` |

Additional units on the same shift count outward from those bases (204, 208 for morning; 206, 209 for afternoon; 214, 217 for night).

## 📂 Adding Stations

Stations are stored in **`stations.csv`** in the repository. You can edit this file directly in GitHub's web editor — no software required.

### CSV Column Reference

| Column | Required | Description | Example |
|---|---|---|---|
| `code` | ✅ | Station callsign prefix (2–5 uppercase letters) | `ESP` |
| `name` | ✅ | Station display name — no "Police Station" needed | `Shepparton` |
| `region` | ✅ | Single letter: `N`, `S`, `E` or `W` | `E` |
| `region_label` | ✅ | Full region name for display | `Eastern` |
| `division` | ✅ | Division name — stations with the same value are grouped | `Goulburn Valley (ED3)` |
| `div_code` | | Short internal division code | `ED3` |
| `psa` | | Police Service Area code | `ESP` |
| `hwp` | | Highway Patrol code servicing this station | `ESP` |
| `ciu` | | CIU code servicing this station | `ESP` |
| `classification` | ✅ | Station type — see table above | `regional_24` |

### Example Row

```csv
ESP,Shepparton,E,Eastern,Goulburn Valley (ED3),ED3,ESP,ESP,ESP,regional_24
```

### ⚠️ Common Mistakes to Avoid

| ❌ Mistake | ✅ Fix |
|---|---|
| Using `Western` as the region key | Use the single letter `W` |
| Extra comma in a row (e.g. `EWO,Wodonga,,E,...`) | Remove the extra comma |
| Missing `div_code` field | Leave it blank but keep the comma — e.g. `,,` |
| Incorrect `classification` spelling | Must be exactly: `metro_24`, `metro_non24`, `regional_24`, `regional_non24`, or `regional_single` |
| Using `N/A` for empty fields | Just leave it blank — `,,` |

> 💡 Every data row must have exactly **9 commas** (10 fields). Count them if something looks wrong.

## 💡 Tips & Tricks

**🎲 Randomised callsigns** — Hit *Generate Lineup* again on the same station and services to get a fresh set of callsign numbers. The tool picks randomly within valid ranges each time.

**🔁 Start Over** — The *↺ Start Over* button at the bottom of the lineup takes you back to Step 1 and clears your selection, ready for a new station.

**📏 Slider defaults** — Default unit counts are based on station classification. A Metropolitan (24 Hours) station starts with more cars than a Regional (Single Member) station. Use the sliders to fine-tune from there.

**🔍 No PSA/HWP/CIU shown?** — The Station Support Links section only appears if those fields are filled in for the selected station in `stations.csv`. Add the relevant codes to make them show up.

**🏍️ Solo motorcycles** — These are deliberately off by default because they're less common in real lineups. Enable them with the solo card that appears after selecting HWP or State Highway Patrol.
